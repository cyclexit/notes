## Block
Use a pair of ` ``` `. <br>
You can specify the language after the first ` ``` `.
For emxample:
```cpp
#include <iostream>

using namespace std;

int main() {
  cout << "Hello world!" << '\n';
  return 0;
}
```
``` java
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello world!\n");
  }
}
```
